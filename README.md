# Life Expectancy Classifier

A machine learning application that predicts life expectancy using Logistic Regression and Decision Tree models.

## Features

- **Dual Model Prediction**: Compare Logistic Regression and Decision Tree predictions
- **Real-time Validation**: Input validation and error handling
- **Probability Visualization**: See model confidence scores
- **RESTful API**: Clean API endpoints with full documentation
- **Modern UI**: Responsive design with intuitive interface
